# Icônes de l'application PedaClic Pro

Pour une version de production avec icônes personnalisées, ajoutez :

- **icon.ico** — Windows (256×256 ou multi-résolutions)
- **icon.icns** — macOS (1024×1024)
- **icon.png** — fallback / Linux (512×512 minimum)

Outils recommandés :
- [electron-icon-builder](https://www.npmjs.com/package/electron-icon-builder) : génère .ico et .icns depuis un PNG
- Ou utilisez des convertisseurs en ligne (png → ico, png → icns)
